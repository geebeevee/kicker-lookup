A lookup tool for the kicker music league.

this reads published google drive spreadsheet and allows the following lookup tools:

1. search for bands/artists. when selected will show :
all songs entered
what league
if any of them were "top hits" or "Low misses" from the resepective tab on the spreadsheet
try to find a wikipedia link for the artist and display

2. top submitted artists - will display all artists equal or exceeded the max submission limit (read from worksheet cell)

3. song search - will search for all times a song was submitted. can also display all songs with a given word or text by hitting "enter" after partial search
